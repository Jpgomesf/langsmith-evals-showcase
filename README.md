# langsmith-evals-showcase

A multi-scenario "eval gym" demonstrating the full surface of **LangSmith evaluation** —
heuristic, custom-code, LLM-as-judge, summary, pairwise and trajectory evaluators across
five small apps-under-test, plus online evals, annotation queues, and a CI regression gate.

> Polished README is generated as part of the build — see `docs/EVALUATION_STRATEGY.md`
> for the decision guide on choosing an eval strategy.
